## 📖 关于这本书

本书系统梳理 AI Agent 的理论基础、架构设计与工程实践，从 ReAct 到 Multi-Agent 系统，从学术前沿到生产落地。

### 核心特色

- ✅ **理论 + 实践** — 每个概念配可运行代码
- ✅ **持续更新** — 跟踪 arXiv 最新论文 + GitHub 热门项目
- ✅ **中文友好** — 面向中文开发者，术语中英对照
- ✅ **技能沉淀** — 每章结束后提炼可复用 Skill
